# ネットワーク設定

## 目的
- ホスト名を `RPI-CTRL-01` に変更する。
- IPアドレスを `192.168.0.10` に固定する。

## 前提
- Debian GNU/Linux 13 (trixie)
- NetworkManager を使用している構成を想定する。

## 手順
### 1. 現状確認
```bash
hostname
ip -br address
nmcli device status
```

### 2. ホスト名の変更
```bash
sudo hostnamectl set-hostname RPI-CTRL-01
```

`/etc/hosts` のホスト名も更新する。
```bash
sudo nano /etc/hosts
```

以下のように修正する。
```
127.0.0.1   localhost
127.0.1.1   RPI-CTRL-01
```

### 3. IPアドレスの固定化
接続名を確認する。
```bash
nmcli -t -f NAME,DEVICE,TYPE,STATE con show --active
```

接続名を `CON_NAME` として、IPアドレスを設定する。
```bash
sudo nmcli con modify "CON_NAME" \
  ipv4.addresses 192.168.0.10/24 \
  ipv4.gateway 192.168.0.1 \
  ipv4.dns "192.168.0.1" \
  ipv4.method manual
```

設定を反映する。
```bash
sudo nmcli con down "CON_NAME"
sudo nmcli con up "CON_NAME"
```

### 4. 反映確認
```bash
hostname
ip -br address
```

## 補足
- `ipv4.gateway` や `ipv4.dns` は実環境に合わせて変更する。
- NetworkManager を使用していない場合は、利用しているネットワーク管理方法に合わせて設定する。
